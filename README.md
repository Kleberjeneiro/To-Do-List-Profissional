# To-Do-List-Profissional
# TaskPro - To-Do List Profissional  Projeto de uma aplicação web de lista de tarefas desenvolvida com **HTML, CSS e JavaScript**.

## Objetivo do projeto

O objetivo do TaskPro é demonstrar conhecimentos fundamentais de desenvolvimento front-end, organização de código, manipulação do DOM, armazenamento de dados no navegador, responsividade e criação de interfaces modernas.

Este projeto pode ser apresentado no GitHub e usado como parte de um portfólio para candidaturas a estágio, suporte técnico, desenvolvimento web júnior ou áreas relacionadas à tecnologia.

## Funcionalidades

- Tela de login simulada
- Salvamento do utilizador no navegador
- Cadastro de tarefas
- Categorias de tarefas
- Filtros por categoria
- Marcar tarefa como concluída
- Excluir tarefa
- Estatísticas de tarefas:
  - total de tarefas
  - tarefas concluídas
  - tarefas pendentes
- Drag and drop para reordenar tarefas
- Modo escuro
- Layout responsivo para computador, tablet e telemóvel
- Salvamento das tarefas com LocalStorage

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- DOM Manipulation
- Drag and Drop API
- Responsividade com Media Queries

## Como executar o projeto

1. Faça o download ou clone este repositório.
2. Abra a pasta do projeto.
3. Clique duas vezes no ficheiro `index.html`.
4. O projeto será aberto no navegador.

Também pode usar a extensão **Live Server** no VS Code.

## Como usar

1. Digite um nome de utilizador.
2. Digite qualquer palavra-passe.
3. Adicione uma nova tarefa.
4. Escolha uma categoria.
5. Clique em **Adicionar**.
6. Use os filtros para visualizar tarefas por categoria.
7. Arraste as tarefas para alterar a ordem.
8. Use o botão de modo escuro para trocar o tema.

## Observação sobre o login

Este projeto usa um login demonstrativo com LocalStorage. Não existe autenticação real com servidor ou banco de dados externo.

Para uma versão mais avançada, o projeto pode ser melhorado com:

- Node.js
- Express
- MySQL
- Firebase
- Supabase
- autenticação real
- API REST
